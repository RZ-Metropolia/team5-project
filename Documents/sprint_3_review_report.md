# Sprint 3 Review Report

1. Extend the functional prototype with additional core features.
2. Integrate Jenkins to establish a CI/CD pipeline.
3. Improve automated testing and code coverage.
4. Create and test a Docker image locally to prepare for containerized deployment.

1. Extended the functional prototype by implementing additional system features and improving backend logic.
2. Integrated the frontend and backend components to support complete user workflows.
3. Set up a Jenkins CI/CD pipeline to automate the build and testing process.
4. Configured the pipeline to run unit tests using JUnit whenever new code is committed.
5. Integrated JaCoCo to generate code coverage reports for the project.
6. Expanded the unit test suite to cover newly implemented functionalities.
7. Built a Docker image locally using Docker and verified that the application runs successfully inside the container.
8. Updated project documentation and repository configuration.
