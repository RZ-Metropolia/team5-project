# Sprint 4 Review Report

## Sprint Goal
1. Finalize the functionality of the prototype.
2. Containerize the application using Docker.
3. Create and publish the Docker image to Docker Hub.
4. Share the functional prototype with the class and prepare for the final demonstration.

## Completed Tasks
1. Finalized the core functionality of the system including authentication, assignment management, submission handling, and grading.
2. Fixed bugs related to data display, user interface behavior, and error handling.
3. Performed testing to verify business logic, user interface responses, and correct data storage and retrieval.
4. Created a Dockerfile to containerize the entire application.
5. Built the Docker image using the Docker build command and verified that the container runs correctly.
6. Tagged and pushed the Docker image to Docker Hub, making it publicly accessible.
7. Shared the functional prototype with classmates and prepared the final presentation materials.
8. Updated project documentation, including README, architecture description, and sprint reports in the GitHub repository

## Demo Summary
The following things are presented during the sprint 4 review:
1. The final working version of the Homework Submission and Tracking System.
2. A demonstration of the main user flows for both teachers and students.
3. The Docker image creation process and Docker Hub repository link.
4. Running the application using the Docker container.
5. The updated project documentation and repository structure.

## What Went Well
1. The system functionality was successfully completed and integrated into a working prototype.
2. Docker containerization made the application easier to run and deploy.
3. Team collaboration and task division helped the team complete the final sprint efficiently.

## What could be improved
1. Improve the user interface and user experience in future iterations.
2. Add more automated testing to further improve system reliability.
3. Enhance system scalability and deployment options, such as cloud deployment.
