# Teacher
1. As a Teacher, I want to log in with a teacher account so that I can use the grading tools.
1. As a Teacher, I want to create a new assignment with a title, description, and due date so that students know exactly what to submit.
1. As a Teacher, I want to click a button to open or download a student's uploaded file so that I can read their work.
1. As a Teacher, I want to see a list of all students who have submitted work for a specific assignment so that I can see who is finished.
1. As a Teacher, I want to input a numeric grade and comments so that the student receives their assessment.
1. As a Teacher, I want to edit the due date or description of an existing assignment so that I can correct mistakes or extend deadlines.
1. As a Teacher, I want to delete an assignment so that I can remove tasks I added by mistake.
1. As a Teacher, I want to change a grade I already gave so that I can fix a mistake if I graded it wrong.
1. As a Teacher, I want to see a list of ungraded assignments ordered by uploaded date so that I can grade them.
1. As a Teacher, I want to see a list of ungraded assignments categorized by course name so that I can easily grade many assignments for one specific course.
1. As a Teacher, I want to save the class grades into a text file (CSV) so that I can open them in Excel or print them.
1. As a Teacher, I want to require specific file types (like PDF only) so that students only send the required type of file.
1. As a Teacher, I want to archive outdated courses so that I can focus on current courses.
1. As a Teacher, I want to save an assignment as a 'Draft' without showing it to students so that I can finish writing the instructions later.
# Student
1. As a Student, I want to log in with my own username and password so that the workspace is linked to me.
1. As a Student, I want to see a list of all open assignments so that I can select which one to work on.
1. As a Student, I want to select a file from my computer and upload it so that my teacher receives my completed work.
1. As a Student, I want to see the grade and comments next to my submitted assignment so that I can see how I performed.
1. As a Student, I want to see a visual alert if a deadline is within 24 hours so that I can prioritize that task.
1. As a Student, I want to see my teacher's email address so that I can contact them if I have a question.
1. As a Student, I want to be told if my file is too big to upload so that I don't waste time waiting for it to fail.
1. As a Student, I want to archive outdated courses so that I can focus on current courses.
1. As a Student, I want to see my due dates on a calendar picture instead of a list so that I can plan my week better.
1. As a Student, I want to save an assignment as a 'Draft' without showing it to teachers so that I can finish the assignments later.
